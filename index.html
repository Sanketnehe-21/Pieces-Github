Your First GitHub Project: A Step-by-Step Guide

Welcome!

Ready to take your first steps into the world of version control and code sharing? This guide will walk you through the process of creating a simple project and uploading it to GitHub.

Don't worry, we'll take it one step at a time!

Let's Get Started

  1. Download and Install Git

    First things first, you'll need Git installed on your computer. Head over to the official Git website and download the version for your operating system.

    Once downloaded, run the installer and follow the on-screen instructions. You'll be up and running with Git in no time!

  2. Create a GitHub Account

    GitHub is where the magic of code sharing happens! If you don't already have an account, visit GitHub and sign up for a free account.

  3. Set Up Your Local Project

    Create a new folder on your computer where you'll keep your project files. Give it a descriptive name, like "my-first-github-project".

    Inside the folder, create three files:

    • index.html (for your website's content)
    • style.css (for styling the look of your website)
    • script.js (for adding interactivity to your website)

  4. Basic HTML Structure

    Open index.html in your code editor and paste in the following basic HTML structure:

    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>My First Website</title>
      <link rel="stylesheet" href="style.css">
    </head>
    <body>
      <script src="script.js"></script>
    </body>
    </html>

  5. Add Some CSS Styling

    Open style.css and let's add a bit of color:

    body {
      background-color: lightblue;
    }

  6. A Touch of JavaScript

    Finally, open script.js and add a simple console message:

    console.log("Hello, GitHub!");
